# 档案AI共学社 - 非遗知识互动平台

## 产品定位
**档案AI共学社**是在互联网和AI时代的个人名片，定位为：
- 📚 知识面广的非遗文化知识平台
- 💬 方便互动的AI交流社区
- 🎨 界面精美的文化展示窗口
- 🔗 多平台统一品牌账号

## 核心功能（非图片生成导向）

### 1. AI 非遗知识助手
- 基于 DeepSeek API 的智能问答
- 覆盖十大门类1557个非遗项目
- 多轮对话，深度交流

### 2. 非遗知识库
- 国家级非遗项目名录（3610个子项）
- 十大门类分类浏览
- 项目详情、历史渊源、传承故事

### 3. 互动社区
- 用户提问、AI回答
- 知识分享、讨论交流
- 学习记录、收藏功能

### 4. 品牌展示
- 档案AI共学社品牌露出
- 作者介绍、联系方式
- 多平台账号引导

## 技术架构
- Next.js 16 + React 19
- SQLite 轻量级数据库
- DeepSeek API 接入
- 静态导出部署
